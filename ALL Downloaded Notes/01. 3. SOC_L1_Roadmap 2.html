PHASE 1: THE FOUNDATION

Before you touch a SIEM, you must understand the battlefield. The SOC is the central nervous system of an organization's defense.

Why SOC Exists?

Organizations bleed data and money. The SOC exists to minimize "Dwell Time"—the time an attacker stays hidden. Without SOC, breaches are discovered by customers or the FBI, causing massive reputational and financial loss.

  • ⚠ Problem: 24/7 Global Cyber Attacks.
  • ✓ SOC Solution: Continuous Monitoring & Rapid Triage.

Operational Focus: SOC vs NOC vs IR

Beginners often confuse these roles. The NOC fixes servers; the SOC hunts hackers; IR cleans up the mess.

PHASE 3: THE ARSENAL

You don't need to code like a dev, but you must query like a detective. These are the weapons you must master.

Required Proficiency Level (0-10)

SIEM (The Brain)

Splunk, Sentinel, QRadar. It collects logs.
Task: Learn SPL (Splunk) or KQL (Sentinel). You must know how to filter: `index=firewall src_ip=10.10.10.5 action=blocked`

EDR (The Hands)

CrowdStrike, Defender, SentinelOne. It watches the endpoint.
Task: Check process tree. Did `excel.exe` spawn `powershell.exe`? That is malicious.

Ticketing (The Logbook)

ServiceNow, Jira.
Task: Write clear, concise notes. "User downloaded file X. Hash matches Malice. Host isolated."

Top Interview Scenarios

Q: "You see a login from China. What do you do?"

Answer: Don't panic. Context is King. 1) Check user location history (is this normal?). 2) Check if user is on travel. 3) Check successful vs failed. 4) Contact user/manager to verify.

Q: "Difference between Encoding, Encryption, Hashing?"

Answer: Encoding = Usability (Base64, reversible). Encryption = Confidentiality (AES, requires key). Hashing = Integrity (SHA256, one-way).

Q: "Describe the port numbers for DNS, HTTP, HTTPS, SSH."

Answer: DNS (53 UDP/TCP), HTTP (80), HTTPS (443), SSH (22). Know your common ports!
